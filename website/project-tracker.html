Project Notes

  • Synergy with TV Enclosures: Same manufacturing partner (Al Shurooq), similar materials, shared brand (Apex Enclosures).
  • User Context: User has semi-sunken pump enclosure already (quiet). Heater/chiller is the noise problem requiring acoustic solution.
  • Priority: APE-HC1 (acoustic) is the primary product - solves a real pain point. APE-P1 (pump) is secondary.
  • Acoustic Louvres: Key component - need to source or design custom. This is the main technical challenge.
